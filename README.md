# User Registration & Management System

A simple web app built with HTML, CSS and Vanilla JavaScript.

---

## 👤 User Features

- Register with name, age, role, email, description and profile image
- Form validations — empty fields, valid email, age 18+, duplicate email check
- Profile image preview on card after registration
- User card displayed after successful registration

## 🔐 Admin Features

- Secure login with admin ID and password
- View all registered users as cards
- Edit any user — form prefills with user data and highlights with animation
- Remove any user instantly
- Search users by name in real time
- Live total user count display

---

## 🛠️ Tech Used

**HTML & CSS**

- Semantic HTML5 form elements
- CSS Flexbox for layout
- CSS animations — shake and glow highlight effect on edit
- Hover effects and smooth transitions on cards and inputs

**JavaScript**

- DOM manipulation — `createElement`, `appendChild`, `classList`
- `async/await` and `Promise` for image file reading
- `FileReader` API — converts image to base64 for storage
- `localStorage` with `JSON.stringify` and `JSON.parse` for data persistence
- `Array.filter`, `Array.findIndex`, `Array.some` for data operations
- Regex for email format validation
- `debounce` function to optimise search performance
- `scrollIntoView` for smooth scroll to form on edit

---
